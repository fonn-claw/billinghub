1. Create `src/lib/auth/session.ts` with iron-session v8:
   ```typescript
   import { getIronSession } from "iron-session";
   import { cookies } from "next/headers";

   export interface SessionData {
     userId: string;
     email: string;
     role: "manager" | "billing_clerk" | "customer";
     name: string;
   }

   export const sessionOptions = {
     password: process.env.SESSION_SECRET!,
     cookieName: "billinghub-session",
     cookieOptions: {
       secure: process.env.NODE_ENV === "production",
       httpOnly: true,
       sameSite: "lax" as const,
       maxAge: 60 * 60 * 24 * 7, // 7 days
     },
   };

   export async function getSession() {
     const cookieStore = await cookies();
     return getIronSession<SessionData>(cookieStore, sessionOptions);
   }
   ```
   CRITICAL: `await cookies()` -- Next.js 15 async API.
   Export sessionOptions separately (middleware needs it).

2. Create `src/lib/auth/actions.ts` as "use server" module:
   - `login(formData: FormData)`: Extract email and password. Query users table with `eq(users.email, email.toLowerCase())`. If no user or bcrypt.compare fails, return `{ error: "Invalid email or password" }`. On success, set session fields (userId, email, role, name), call session.save(). Redirect: customer -> "/portal", else -> "/dashboard".
   - `logout()`: Call getSession(), session.destroy(), redirect("/login").

3. Create `src/middleware.ts` for route protection:
   - Import getIronSession from "iron-session" and SessionData type.
   - Duplicate sessionOptions inline (middleware runs in edge runtime, cannot import from session.ts which uses next/headers).
   - Public routes: /login, /_next, /assets, /favicon.ico -- return NextResponse.next().
   - No session (no userId) -> redirect to /login.
   - Customer role + path not starting with /portal -> redirect to /portal.
   - Non-customer role + path starts with /portal -> redirect to /dashboard.
   - Use `getIronSession<SessionData>(req, res, sessionOptions)` (req/res signature for middleware).
   - Config matcher: `["/((?!_next/static|_next/image|favicon.ico|assets/).*)"]`
