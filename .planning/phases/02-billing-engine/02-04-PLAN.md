Reject if invoice status is "draft" (cannot pay a draft invoice). Insert payment. Then compute new status using computeInvoiceStatus(invoice.status, totalCents, existingPaidCents + amountCents, dueDate) and UPDATE invoice status. revalidatePath("/payments"), revalidatePath(`/invoices/${invoiceId}`), revalidatePath(`/customers/${invoice.customerId}`). **1. Create `src/lib/dal/recurring.ts`:**

- `getTemplates()`: Query recurringTemplates JOIN customers (for name). Return: id, name, customerName, customerId, frequency, nextInvoiceDate, active, lineItems (parsed from JSONB), createdAt. Order by name.

- `getTemplate(id: string)`: Single template by id with customer info.

- `getDueTemplates()`: Query active templates WHERE nextInvoiceDate <= today. Used for bulk generation.

**2. Create `src/app/(admin)/recurring/actions.ts`:**

- `createTemplate(prevState: unknown, formData: FormData)`: Parse JSON from formData.get("data"), validate with createRecurringTemplateSchema, check session role. Insert into recurringTemplates with lineItems stored as JSONB (array of `{ category, description, quantity, unitPriceCents }` objects). revalidatePath("/recurring"). redirect to /recurring.

- `toggleTemplate(formData: FormData)`: Read templateId from formData. Toggle `active` field (flip current value). revalidatePath("/recurring").

- `generateInvoice(formData: FormData)`: Read templateId from formData. Load template. Create a new draft invoice from template data: use template's customerId, lineItems (compute subtotal/tax/total from items, use default 8.5% tax rate), set issueDate to template's nextInvoiceDate, dueDate to issueDate + 30 days. Get next invoice number. Insert invoice + line items. Advance template's nextInvoiceDate by frequency period: monthly = addMonths(1), quarterly = addMonths(3), annual = addYears(1). Use date-fns `addMonths`/`addYears` and `format(date, "yyyy-MM-dd")`. revalidatePath("/recurring"), revalidatePath("/invoices"). Return `{ success: true, invoiceId }`.

- `bulkGenerate(prevState: unknown, formData: FormData)`: Call getDueTemplates(). For each template, run the same logic as generateInvoice. Return `{ success: true, count: N }` where N is number of invoices generated.
