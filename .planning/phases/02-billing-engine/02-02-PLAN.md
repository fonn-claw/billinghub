**1. Create `src/lib/dal/customers.ts` with these functions:**

- `getCustomers(search?: string, dock?: string)`: Query all customers, LEFT JOIN to invoices to compute outstanding balance. If `search` provided, filter by `ilike` on name or email. If `dock` provided, filter by `eq(customers.dock, dock)`. Return array sorted by name. Each row includes: id, name, email, phone, dock, slipNumber, vesselName, outstanding balance (sum of invoice totals minus sum of payments for non-draft invoices).

- `getCustomer(id: string)`: Single customer by id. Return null if not found.

- `getCustomerBalance(customerId: string)`: Aggregate query returning `{ totalCharged, totalPaid, currentBalance, overdueAmount }`. All in integer cents. totalCharged = SUM(invoices.totalCents) WHERE status != 'draft'. totalPaid = SUM(payments.amountCents) for that customer's invoices. currentBalance = totalCharged - totalPaid. overdueAmount = SUM of (totalCents - paid) for invoices WHERE dueDate < CURRENT_DATE AND status NOT IN ('draft','paid').

- `getCustomerNotes(customerId: string)`: Query customerNotes joined with users (for createdBy name), ordered by createdAt DESC.

- `getCustomerInvoices(customerId: string)`: Query invoices WHERE customerId, LEFT JOIN payments for paid amount, ordered by issueDate DESC. Return with computed balance and status.

- `getCustomerPayments(customerId: string)`: Query payments JOIN invoices WHERE invoices.customerId, ordered by paymentDate DESC. Include invoice number in result.

**2. Create `src/app/(admin)/customers/actions.ts`:**

- `createCustomer(prevState: unknown, formData: FormData)`: Parse JSON from formData.get("data"), validate with createCustomerSchema, check session for manager/billing_clerk role, insert into customers table, revalidatePath("/customers"), return `{ success: true, customerId }`.

- `updateCustomer(prevState: unknown, formData: FormData)`: Same validation, update by id. Only allow if session role is manager or billing_clerk. revalidatePath for both /customers and /customers/[id].

- `addCustomerNote(prevState: unknown, formData: FormData)`: Parse `{ customerId, note }` from formData, validate note is non-empty, insert into customerNotes with session.userId as createdBy. revalidatePath(`/customers/${customerId}`).

All actions follow the `(prevState, formData)` signature for useActionState compatibility, matching the pattern in `src/lib/auth/actions.ts`.
