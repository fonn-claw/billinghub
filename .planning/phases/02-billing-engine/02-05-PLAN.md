**1. Create `src/lib/dal/collections.ts`:**

- `getCollections()`: Query customers WHERE isCollectionsFlagged = true. For each flagged customer, compute: totalOverdue (SUM of invoice balances where status is overdue or collections), maxDaysOverdue (MAX days past due across their overdue invoices), lastNoteDate (MAX createdAt from collectionNotes for this customer), lastNotePreview (first 80 chars of most recent collection note). JOIN invoices + payments for overdue calculations, LEFT JOIN collectionNotes for note info. Order by totalOverdue DESC.

  Return array of: `{ id, name, email, phone, dock, slipNumber, totalOverdueCents, maxDaysOverdue, lastReminderDate, lastNoteDate, lastNotePreview, agingBucket }`.

- `getOverdueCustomers()`: Query customers who have at least one overdue invoice (status in ('overdue','partial','sent') AND dueDate < today AND NOT fully paid) but are NOT already flagged. For bulk-flagging UI. Return: id, name, totalOverdueCents, maxDaysOverdue.

- `getCollectionNotes(customerId: string)`: Query collectionNotes WHERE customerId, LEFT JOIN users (for createdBy name). Order by createdAt DESC. Return: id, note, noteType, promisedDate, promisedAmountCents, createdByName, createdAt.

**2. Create `src/app/(admin)/collections/actions.ts`:**

- `toggleCollectionsFlag(formData: FormData)`: Read customerId from formData. Toggle isCollectionsFlagged on customers table. If flagging ON, also update the invoice status of all overdue invoices for this customer to 'collections'. revalidatePath("/collections"), revalidatePath(`/customers/${customerId}`).

- `bulkFlag(formData: FormData)`: Read customerIds (JSON array string) from formData. Set isCollectionsFlagged = true for all specified customers. Update their overdue invoices to 'collections' status. revalidatePath("/collections").

- `addCollectionNote(prevState: unknown, formData: FormData)`: Parse JSON from formData.get("data"), validate with collectionNoteSchema. If noteType is "promise_to_pay", require promisedDate and promisedAmountCents to be non-empty. Insert into collectionNotes with session.userId as createdBy. revalidatePath(`/collections`), revalidatePath(`/customers/${customerId}`). Return `{ success: true }`.

- `updateReminderDate(formData: FormData)`: Read customerId from formData. Set lastReminderDate to today's date (format "YYYY-MM-DD" using `format(new Date(), "yyyy-MM-dd")` from date-fns). revalidatePath("/collections"), revalidatePath(`/customers/${customerId}`). Return `{ success: true }`.

Below or in a separate section: "Unflagged Overdue Accounts" showing overdueCustomers with a "Flag for Collections" button or checkbox for bulk flagging. Only show this section if overdueCustomers is non-empty.

If any overdueCustomers exist, show alert banner: "N customer(s) with overdue invoices are not yet flagged for collections." Render CollectionsTable with both datasets.
