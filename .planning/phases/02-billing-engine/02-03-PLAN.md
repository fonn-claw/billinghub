**1. Create `src/lib/dal/invoices.ts`:**

- `computeInvoiceStatus(currentStatus: string, totalCents: number, paidCents: number, dueDate: string, today: Date = new Date())`: Pure function returning computed status. Logic: if draft return "draft"; if collections return "collections" (sticky); if paidCents >= totalCents return "paid"; if paidCents > 0 return "partial"; if new Date(dueDate) < today return "overdue"; else return "sent".

- `getInvoices(status?: string, search?: string)`: Query invoices LEFT JOIN customers (for name) LEFT JOIN payments (for SUM of amountCents). For each invoice, compute: paidCents (SUM of payments), balanceCents (totalCents - paidCents), effectiveStatus (using computeInvoiceStatus). If `status` filter provided, filter by effectiveStatus (post-query filtering since status is computed). If `search` provided, filter by invoice number or customer name ilike. Order by issueDate DESC. Return array with: id, invoiceNumber, customerName, customerId, issueDate, dueDate, totalCents, paidCents, balanceCents, status (computed), agingBucket (from getAgingBucket for non-draft invoices).

- `getInvoiceWithDetails(id: string)`: Single invoice with all line items and all payments. LEFT JOIN lineItems, LEFT JOIN payments. Compute paidCents, balanceCents, effectiveStatus. Return null if not found. Include: invoice fields, lineItems array, payments array (with payment date, amount, method, reference), paidCents, balanceCents, computed status.

- `getCustomerSelectList()`: Minimal query returning `[{ id, name, dock, slipNumber }]` for the customer selector dropdown, ordered by name.

**2. Create `src/app/(admin)/invoices/actions.ts`:**

- `createInvoice(prevState: unknown, formData: FormData)`: Parse JSON from formData.get("data"), validate with createInvoiceSchema, check session role (manager/billing_clerk). Compute subtotalCents = sum of (qty * unitPriceCents) for all items. Compute taxAmountCents = Math.round(subtotalCents * taxRate). Compute totalCents = subtotalCents + taxAmountCents. Get next invoice number via getNextInvoiceNumber(). Insert invoice, then insert all line items with the invoice id. revalidatePath("/invoices"), revalidatePath(`/customers/${customerId}`). redirect(`/invoices/${invoice.id}`).

- `updateInvoice(prevState: unknown, formData: FormData)`: Same as create but UPDATE existing invoice + DELETE old line items + INSERT new line items. ONLY allow if current status is "draft" — query invoice first and reject with error if status !== "draft". Include invoiceId in form data.

- `markAsSent(prevState: unknown, formData: FormData)`: Update invoice status from "draft" to "sent". Reject if not currently "draft". revalidatePath.

- `deleteInvoice(formData: FormData)`: Delete invoice by id. Only allow if status is "draft". revalidatePath("/invoices").
